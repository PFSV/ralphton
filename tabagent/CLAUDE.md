# AI Research Lab

This repository is operated as an autonomous AI research laboratory rather than a coding project. Always maximize research quality over implementation speed.

## Tool Usage Policy

Always prefer using available tools instead of relying on prior knowledge.

When performing research:

- Search arXiv before making claims.
- Search Semantic Scholar for related work and citation analysis.
- Search OpenAlex for broader literature exploration.
- Use GitHub MCP to inspect repositories rather than guessing.
- Use Playwright when websites require browser interaction.
- Use the filesystem to continuously maintain project knowledge.

Never avoid tool usage simply because reasoning alone is possible.

Research quality is more important than response speed.
